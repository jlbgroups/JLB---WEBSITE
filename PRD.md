# 📄 Product Requirements Document (PRD)

## Project: Shnoor Website Redesign (Minimal Premium Version)

---

## 1. Overview

### 1.1 Purpose

Redesign the existing Shnoor International LLC website to create a **modern, minimal, premium digital presence** that clearly communicates its offerings in:

* IT Solutions
* AI & Data
* Global Trade & Logistics

The new website should improve **clarity, credibility, and lead generation**.

---

### 1.2 Background / Problem Statement

The current website suffers from:

* Mixed positioning (IT + trade not clearly unified)
* Weak visual hierarchy
* Overly text-heavy layout
* Low conversion focus

This creates confusion for potential clients and reduces engagement.

---

### 1.3 Goals

#### Primary Goals

* Clearly communicate value proposition
* Increase inbound leads
* Improve user trust and perception

#### Secondary Goals

* Improve SEO visibility
* Enable scalable content updates
* Create a reusable design system

---

### 1.4 Success Metrics (KPIs)

| Metric               | Target      |
| -------------------- | ----------- |
| Bounce Rate          | ↓ 25%       |
| Avg. Time on Site    | ↑ 40%       |
| Lead Conversion Rate | ↑ 2–3x      |
| Page Load Time       | < 2 seconds |

## 5. Page-Level Requirements

---

### 5.1 Home Page

#### Objective

Provide immediate clarity on what the company does and drive user action.

#### Sections

1. Hero Section

   * Headline: “Technology Meets Global Trade for Limitless Growth”
   * Subtext + 2 CTAs

2. Services Overview

   * 5 cards:

     * IT Solutions
     * AI & Data
     * Cloud & Infrastructure
     * Consulting
     * Global Trade

3. Global Presence

   * Static world map with regions

4. Case Studies (Preview)

   * 2–3 examples with metrics

5. Trust Section

   * Client logos

6. CTA Section

   * “Get a Consultation”

## 7. UX / UI Requirements

### 7.1 Design Principles

* Minimal and elegant
* High whitespace usage
* Clear visual hierarchy
* No clutter

---

### 7.2 Visual Style

#### Colors

* Background: #F7F8F6
* Primary: #1F3D2B
* Accent: #3FA26E

#### Typography

* Headings: Poppins / Inter (600–700)
* Body: Inter (400–500)

---

### 7.3 Components

* Cards (rounded, subtle border)
* Buttons (primary + secondary)
* Grid-based layout
